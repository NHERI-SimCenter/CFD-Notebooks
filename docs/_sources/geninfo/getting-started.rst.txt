Getting started with CFDN
===========================

CFD Notebooks are available as Jupyter notebooks and hosted on Designsafe. The notebooks are almost self-explanatory and does not require the learners to have the knowledge of Python or Jupyter notebooks.

In the below video, it is shown on how to:

- Access the Jupyter notebooks from the Designsafe website
- Learn how to run the files
- Learn how to use Jupyter notebook
- Edit and work with Jupyter notebooks
- Running jobs on TACC
